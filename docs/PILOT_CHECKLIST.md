# Pilot Checklist

## Recommended Pilot Shape

Keep the first pilot small.

Use:

1. one machine
2. one named owner
3. one local LM Studio server
4. the six built-in prompts first
5. text-first tasks

## Before Starting

Confirm:

- the model is loaded locally
- the app opens on localhost
- staff know that nothing is auto-sent
- retention expectations are clear
- the chosen task is narrow and useful

## Built-In Prompt Set

The MVP includes these six prompts in the prompt menu:

1. `E-Mail to Parents/Students`
2. `Extract Text from PDF/Image`
3. `Student Support Assistant`
4. `5E Lesson Plan Creator`
5. `School Newsletter`
6. `Social Media Post Generator`

## Good First Tasks

- draft a calm reply
- summarise key points
- turn notes into action items

## Avoid At First

- broad whole-school rollout
- many prompts at once
- weak governance ownership
- unclear real-data rules
